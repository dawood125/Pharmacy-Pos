Pharmacy Store 
Two Roles: Admin , Cashier
Navbar:  Alert Bell(Low Stock , Expiry), Logout Button
Sidebar: 
1.	POS
2.	Inventory Management
3.	Dashboard
4.	Customer Return
5.	Report

1. 🛒 POS (Point of Sale) — Main Feature(F1 shortcut key)
Flow:
•	Search product (by name / barcode)
•	Medicine Price and Quantity
•	Click → goes to cart
•	Adjust quantity
•	Delete from Cart
•	Auto price + total
•	Checkout →Unique invoice number →generates bill, print Bill and save (ctrl p shortcut key)
•	Stock decreases

Important MVP features:
•	Fast search (debounced input)
•	Keyboard-friendly (for shop speed) 
•	Cash Recived : Cash/Easypaisa / Jazzcash / Banktransfer

2. Stock Entry Inventory
Medicine fields:
•	Name
•	Expiry date
•	Purchase price
•	Sale price
•	Quantity
•	Edit Inventory List
•	Update inventory List
•	Search inventory
1.	Store medicine details (name, price, quantity, expiry date)
2.	Automatically reduce stock when a sale is made
3.	Increase stock when items are returned
3. Dashboard (Simple but Useful)
•	Only 4 things for MVP:
•	Today sales
•	Total profit (approx)
•	Low stock alert
•	Expiring soon medicines

4. Customer Return System (Easy Explanation)
•	Flow:
•	Search invoice
•	Select item
•	Return quantity
•	Stock increases again
1.	Search previous sales
2.	Return specific medicines
3.	Update stock accordingly
5. Report Section
•	Daily sales report
•	Monthly profit List
•	Best-selling medicines List
•	Low stock list
•	Expiry Product List report
•	Print Option to All reports or download Document
